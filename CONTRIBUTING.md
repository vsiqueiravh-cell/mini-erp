# Contributing

This repository is a public portfolio sample maintained by Victor Henrique de Siqueira.

Suggestions, issues and improvements are welcome when they keep the project aligned with its purpose: demonstrating enterprise software architecture with fictitious, public-safe examples.

Please do not submit:

- Proprietary client or employer code
- Real credentials, tokens, URLs or tenant identifiers
- Production customer data
- Company-confidential process details

For questions, contact [victor.h.siqueira@hotmail.com](mailto:victor.h.siqueira@hotmail.com).
